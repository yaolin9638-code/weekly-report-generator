# Weekly Report Generator

Auto-generate weekly reports from Slack messages and publish to Google Sheets.

## Features
- Fetch work updates from Slack channels (web-rnd, seseai-team)
- Generate structured weekly report
- Publish to Google Sheets with formatting preserved
- Send Telegram notification

## Setup
1. Configure Slack Token with groups:history permission
2. Set up Google Service Account credentials
3. Configure Google Sheet template

## Usage
- Trigger: Every Friday 13:30
- Or manual trigger "生成周报"
